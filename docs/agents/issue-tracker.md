# Issue Tracker

Issues for this repo live in **GitHub Issues** at `adwinying/sqlcery`.

## CLI

Use the `gh` CLI for all issue operations:

    gh issue list
    gh issue create --title "..." --body "..."
    gh issue view <number>
    gh issue edit <number> --add-label "needs-triage"
    gh issue close <number>

## Conventions

- Apply the `needs-triage` label when creating a new issue.
- Reference issues in commits and PRs with `Fixes #<number>`.
